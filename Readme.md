# Boilerplate cross-platform Atari 8bit Development 🖥️

This is just a starter project to develop Atari 8bit native programs on either a Mac or a Windows computer
You modify the config.mk file depending on your environment and it will build the code and launch and emulator for testing.

You will need to have Atari800MacX installed and working on the Mac or Altirra for the Windows machine.  
This assumes you have Visual Studio Code and CC65 installed correctly. You will also need make for the Windows machine.
